## smartystreets

#### How to Run using SDK
```
node address-validation.js
```

#### SmartyStreets APIs

```
https://us-street.api.smartystreets.com/street-address
?auth-id=<authId>
&auth-token=<authToken>
&street=1600+amphitheatre+pkwy
&city=mountain+view
&state=CA
&candidates=10
```

```
https://us-zipcode.api.smartystreets.com/lookup
?auth-id=<authId>
&auth-token=<authToken>
&city=mountain+view
&state=CA
&zipcode=94043
&street=1600+amphitheatre+pkwy
```
